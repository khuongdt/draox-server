2. Công Nghệ Sử Dụng

Stack công nghệ được lựa chọn kỹ lưỡng để đảm bảo hiệu suất cao, khả năng mở rộng và trải nghiệm phát triển tốt nhất. Ant Design Pro 6 là nền tảng chính, kết hợp với TypeScript strict và các thư viện chuyên biệt cho dashboard thời gian thực.

React React 18 + Hooks

React 18 với Concurrent Features, Suspense, và toàn bộ lifecycle hooks. Không dùng class components. useCallback và useMemo được áp dụng cho tối ưu hiệu suất rendering.

TypeScript TypeScript 5 Strict

Chế độ strict hoàn toàn — không any, bắt buộc kiểu trả về, null checks nghiêm ngặt. Tất cả API response và props đều có interface rõ ràng.

Ant Design Ant Design 5 + Pro

Bộ component library đầy đủ với ProLayout, ProTable, ProForm, ProDescriptions, ProCard và StepsForm. CSS-in-JS token system cho theming toàn diện.

UmiJS UmiJS 4 Runtime

Framework tích hợp với plugin hệ thống: @umijs/plugins bao gồm antd, locale, access, model, initialState và request. Convention-based routing.

Charts @ant-design/charts

Thư viện biểu đồ chuyên nghiệp: Area (băng thông), Line (kết nối), Column (traffic), Pie (phân bố), Gauge (điểm uy tín IP). Tích hợp trực tiếp với AntD theme.

Mock UmiJS Mock Server

Mock server tích hợp sẵn theo convention mock/*.ts. Hot-reload khi thay đổi file mock. Mô phỏng WebSocket qua polling interval với dữ liệu ngẫu nhiên thực tế.

config/ — Cấu Hình UmiJS

4 tệp cấu hình trung tâm. config.ts khai báo tất cả plugins, webpack alias và proxy rules. routes.ts chứa 22 route với trường access cho RBAC.

mock/ — Dữ Liệu Giả Lập

10 tệp mock theo convention UmiJS. Server tự động load và hot-reload. Không cần cài đặt thêm — chỉ cần đặt tệp vào thư mục mock/.

src/components/ — Thành Phần Tái Sử Dụng

11 component chuyên biệt cho dashboard: biểu đồ, bảng dữ liệu, chỉ báo trạng thái, thanh sức khỏe hệ thống và indicator WebSocket.

src/services/ — Lớp Dịch Vụ

15 tệp service với axios instance chung, Bearer token interceptor và chuẩn hóa API envelope {'{ success, data, message }'}. TypeScript generic đảm bảo type safety.

src/pages/ — 22 Trang Quản Trị

Mỗi trang là module độc lập với props và state riêng. Sử dụng ProTable, ProForm, ProCard từ Ant Design Pro. Phân quyền qua UmiJS access plugin.

src/locales/ — Đa Ngôn Ngữ

10 tệp ngôn ngữ: en-US.ts + vi-VN.ts làm entry point, cùng 4 namespace mỗi ngôn ngữ: menu, pages, component, global. Tổng ~400 chuỗi dịch.

src/access.ts — Kiểm Soát Truy Cập

Hàm access() nhận initialState và trả về object permission. Được dùng trực tiếp trong route config và component <Access>.

src/app.tsx — Runtime UmiJS

Khai báo getInitialState, layout, request runtime configs. Điểm khởi tạo JWT interceptor và theme token override.

Chuẩn API Envelope và Interceptor

// services/typings.d.ts
export interface ApiResponse<T = unknown> {
  success: boolean;
  data: T;
  message?: string;
  errorCode?: string;
}

// app.tsx — axios request interceptor
export const request: RequestConfig = {
  requestInterceptors: [
    (config) => {
      const token = localStorage.getItem('draox_token');
      if (token) config.headers!['Authorization'] = `Bearer ${token}`;
      return config;
    },
  ],
  responseInterceptors: [
    (response) => {
      const { data } = response;
      if (!data.success) {
        if (data.errorCode === 'UNAUTHORIZED') history.push('/login');
        throw new Error(data.message);
      }
      return response;
    },
  ],
};

8. Thành Phần Tái Sử Dụng

BandwidthChart

Biểu đồ Area hiển thị băng thông inbound/outbound theo thời gian thực. Sử dụng @ant-design/charts Area với dual axis. Nhận dữ liệu qua props với interval polling 3 giây.

Real-time Area Chart

ConfirmActionModal

Modal xác nhận cho các thao tác nguy hiểm (ngắt kết nối, xóa plugin, ban IP). Có trường nhập xác nhận tên/ID để tránh thao tác nhầm. Hỗ trợ custom warning message.

Modal TypeScript

ConnectionTable

Bảng ProTable hiển thị danh sách kết nối đang hoạt động. Lọc theo giao thức, trạng thái, IP. Actions: ngắt kết nối, xem chi tiết. Cập nhật tự động mỗi 5 giây.

ProTable Auto-refresh

DarkStatisticCard

EventTimeline

Timeline sự kiện hệ thống theo thứ tự thời gian ngược. Màu sắc theo mức độ: lỗi (đỏ), cảnh báo (vàng), thành công (xanh), thông tin (xám). Hỗ trợ filter theo loại.
